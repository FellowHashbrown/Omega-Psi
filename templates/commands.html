page.code();

All things having to do with coding go here

  • convert Let's you convert signed or unsigned numbers from one base to another.
    • signed
    • unsigned
  • execute Let's you run code from a slew of languages.
    • C# Run code using the C# (mono 6.6.0.161) language.
    • bash Run code using the Bash (5.0.0) language.
    • basic Run code using the Basic (FBC 1.07.1) language.
    • c Run code using the C (GCC 9.2.0) language.
    • c++ Run code using the C++ (GCC 9.2.0) language.
    • fortran Run code using the Fortran (GFortran 9.2.0) language.
    • go Run code using the Go (1.13.5) language.
    • haskell Run code using the Haskell (GHC 8.8.1) language.
    • java Run code using the Java (OpenJDK 13.0.1) language.
    • javascript Run code using the JavaScript (nodejs 12.14.0) language.
    • pascal Run code using the Pascal (FPC 3.0.4) language.
    • php Run code using the PHP (7.4.1) language.
    • prolog Run code using the Prolog (GNU Prolog 1.4.5) language.
    • python Run code using the Python (3.8.1) language.
    • ruby Run code using the Ruby (2.7.0) language.
    • rust Run code using the Rust (1.40.0) language.
  • logic Allows you to get a truth table for a logical expression and the expression simplified.
  • qrCode Create a QR code from given text.
  • textTo Converts text to some encoding options
    • base64 Let's you encode or decode text in Base64 encoding.
      • decode Decode Base64 text into regular text.
      • encode Encode regular text into Base64.
    • binary Lets you encode/decode text into binary.
      • decode Decodes binary into text using ASCII values
      • encode Encodes text into binary using ASCII values
    • decimal Lets you encode or decode text into decimal values
      • decode Decodes decimal values into text using ASCII values
      • encode Encodes text into decimal values using ASCII values
    • hex Lets you encode/decode text into hexadecimal
      • decode Decodes hex into text using ASCII values
      • encode Encodes text into hex using ASCII values
    • morse Shows you a conversion chart for Morse code or lets you encode/decode text and Morse code.
      • decode Decode Morse code into regular text.
      • encode Encode regular text into Morse code.
    • octal Lets you encode/decode text into octal.
      • decode Decodes octal into text using ASCII values
      • encode Encodes text into octal using ASCII values

page.math();

Mathematical stuff like calculus and basic algebra!

  • absolute Attempts to get the absolute value of a value.
  • arccosine Attempts to get the inverse cosine of a value.
  • arcsine Attempts to get the inverse sine of a value.
  • arctangent Attempts to get the inverse tangent of a value.
  • areaUnderCurve Attempts to find the area underneath a curve.
  • cosine Attempts to get the cosine of a value.
  • derivative Attempts to take the derivative of a mathematical expression.
  • factor Attempts to factor a mathematical expression.
  • integral Attempts to integrate a mathematical expression.
  • logarithm Finds the logarithm of a number. Default base is 10
  • matrix Gives you information about a matrix you enter. The format is tuples separated by semi-colons: (1, 2, 3); (4, 5, 6)
  • simplify Simplifies a mathematical expression.
  • sine Attempts to get the sine of a value.
  • tangent Attempts to get the tangent of a value.
  • tangentLine Attempts to find the tangent line of a function at a given value x.
  • zeroes Attempts to find the zeroes of a mathematical expression.

page.music();

A way to listen to music!

  • join Forces Omega Psi to join whichever voice channel you're in!
  • leave This command can only be run by people who have manage server permissions in a server.
    		Leaves the voice channel and clears your queue of songs
  • loop Allows you to loop the current song. Call it again to stop looping it!
  • now Shows you the current song that is playing
  • pause This command can only be run by people who have manage server permissions in a server.
    		Pauses the current song
  • play Plays a song you request!
  • playlist Lets you create, view, or play a playlist saved in the server!
    • add This command can only be run by people who have manage server permissions in a server.
      		Add a song to a playlist you specify!
    • create This command can only be run by people who have manage server permissions in a server.
      		Lets you create a playlist to save to play in the future!
    • delete This command can only be run by people who have manage server permissions in a server.
      		Deletes a specified playlist from your server.
    • play Plays a playlist you specify!
    • remove This command can only be run by people who have manage server permissions in a server.
      		Remove a song from a playlist you specify!
    • view Lets you view the saved playlists in this server!
  • queue Show the queue of which songs are next!
  • remove Allows you to remove a song from the queue
  • resume This command can only be run by people who have manage server permissions in a server.
    		Resumes playing the current song
  • shuffle Shuffles the current queue!
  • skip Skips the currently playing song. If you were not the requester of the song, there will be a vote where you need 3 people to skip it
  • stop This command can only be run by people who have manage server permissions in a server.
    		Stops playing music and clears the queue
  • summon This command can only be run by people who have manage server permissions in a server.
    		Summons the bot to a voice channel. If there was none specified, it'll join yours!
  • volume This command can only be run by people who have manage server permissions in a server.
    		Sets the volume of the music!
